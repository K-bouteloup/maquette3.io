Qovery Engine

Abstraction layer to deploy microservices applications on any Cloud provider

Qovery Engine is an open-source abstraction layer product that turns apps easy deployment on AWS,Azure and others Cloud providers.

How does it work?

Qovery Engine is an open-source abstraction layer product that turns easy apps deployment on AWS,GCP,Azure and other Cloud providers.The Qovery Engine is coded in Rust and takes advantage of terraform,Helm,Kubectl,Docker to manage resources.

Usage

Initialize EKS(AWS Kubernetes) and ECR (AWS container registry) on AWS

Deploy real world applications

Qovery Engine empowers you to deploy complex applications,such as backend,a frontend,and a database in a very simple way.

Servers,networking,securité,all is done bgy the Engine for you.

Resimiency is key

Qovery Engine knows when something goes wrong on deployment of your applications.Qovery Engine is build with resiliency in mind.

A transactional engine inspired by what is provided into ACID databases has been at developed at the heart of the product to rollback on consistent and working application version when something goes wrong.
